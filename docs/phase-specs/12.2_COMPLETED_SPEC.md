## Edge Cases & Rules
1. **Decoupled Scoring:** Match events NEVER auto-calculate scores. Scores are manually typed.
2. **Unique Awards:** Changing MVP/DOTD selection deletes old event, inserts new event.
3. **Card Events:** Yellow/red cards stored as `event_type` with `points: 0`.
4. **Data Consistency:** Fetch `week_teams`, `match_events`, `team_selections` concurrently.
5. **Empty States:** Show "No past weeks" message on Results index.
