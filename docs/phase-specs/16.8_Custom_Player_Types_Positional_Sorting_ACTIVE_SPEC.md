# ACTIVE SPEC: Phase 16.8 - Custom Player Types & Positional Sorting

## 🎯 Why This Matters
Different clubs use different naming conventions for their squads (e.g., "Prem" instead of "Performance", "Development" instead of "Open"). We need to make Player Types customizable within `club_settings` while maintaining clean, beautiful code.

Additionally, the Selection Board currently sorts available players alphabetically, which is inefficient for rugby selection. We must update the Selection Board to sort players:
1. **First** by their custom Player Type order (defined in club settings)
2. **Second** by strict Rugby Positional Order (Props → Hookers → Locks → etc.)
3. **Third** alphabetically (as a fallback)

**Critical Constraint**: The UI must remain a flat, continuous list with NO visual section dividers or headers.

### Step 5: Selection Board Sorting Logic (Core Feature)
Update `src/hooks/useSelectionBoard.ts`:

```typescript
// Import existing constants
import { RUGBY_POSITION_ORDER } from '../lib/supabase';

// Enhanced sorting function
function sortAvailablePlayers(players: Player[], clubSettings: ClubSettings | null) {
  const typeOrder = clubSettings?.player_types || DEFAULT_PLAYER_TYPES;
  
  return [...players].sort((a, b) => {
    // 1. Sort by Player Type (custom club order)
    const typeIndexA = typeOrder.indexOf(a.player_type);
    const typeIndexB = typeOrder.indexOf(b.player_type);
    
    if (typeIndexA !== typeIndexB) {
      if (typeIndexA === -1) return 1;      // Unknown types go last
      if (typeIndexB === -1) return -1;     // Known before unknown
      return typeIndexA - typeIndexB;       // Custom order
    }
    
    // 2. Sort by Rugby Position (using existing RUGBY_POSITION_ORDER)
    const posIndexA = RUGBY_POSITION_ORDER.indexOf(a.primary_position || 'Unspecified');
    const posIndexB = RUGBY_POSITION_ORDER.indexOf(b.primary_position || 'Unspecified');
    
    if (posIndexA !== posIndexB) {
      if (posIndexA === -1) return 1;       // Unspecified positions go last
      if (posIndexB === -1) return -1;      // Specified before unspecified
      return posIndexA - posIndexB;         // Rugby positional order
    }
    
    // 3. Sort Alphabetically (fallback)
    return a.name.localeCompare(b.name);
  });
}
```
