## üßÝ Token Mapping Strategy

| Old Value | New Value | Rule |
|-----------|-----------|------|
| `#6B21A8` | `#0062F4` or `bg-brand`/`text-brand` | Primary brand color |
| `#581C87` | `#7B2FFF` or `bg-brand-dark`/`text-brand-dark` | Dark variant |
| `#F3E8FF` | `#E8F0FE` or `bg-brand-light` | Light variant |
| `#9333EA` | `#0062F4` or `bg-brand` | Loading state (flattened to primary) |
| `rgba(107,33,168,0.15)` | `rgba(0,98,244,0.15)` | Transparent active filter |
| `purple-50` | `bg-brand-light` | Light background |
| `purple-100` | `bg-brand-light` | Light background |
| `purple-200` | `text-brand-light` | Muted text on dark |
| `purple-300` | `text-brand-light` or `focus:ring-brand-light` | Muted text / focus rings |
| `purple-400` | `text-brand-dark` | Icon colors |
| `purple-500` | `ring-brand` | Focus ring |
| `purple-600` | `bg-brand` / `text-brand` | Primary buttons, active text |
| `purple-700` | `bg-brand-dark` / `text-brand-dark` | Toggle switches, dark buttons |
| `purple-800` | `bg-brand-dark` / `text-brand-dark` | Sidebar, spinners, hover states |
| `purple-900` | `bg-brand-dark` | Sidebar background |

### Refactoring Priority for Inline Styles

When encountering `style={{ color: '#6B21A8' }}`:
1. **Best**: Convert to Tailwind className (e.g., `className="text-brand"`)
2. **Good**: Replace hex with CSS variable (e.g., `color: 'var(--primary)'`)
3. **Acceptable**: Replace hex with new hex (e.g., `color: '#0062F4'`)

## ‚öÝÔ∏è Risks & Mitigations

### Risk 1: `AvailabilityForm.tsx` uses dynamic `primary_color` from DB
- **Mitigation**: Fallback default changes from `#6B21A8` to `#0062F4`. Existing clubs with `#6B21A8` stored in DB will still show purple until they update settings.

### Risk 2: Inline DOM manipulation in SelectionBoard
- **Lines 1023-1024** use `e.currentTarget.style.background = '#6B21A8'` ‚Äî these must use the new hex `'#0062F4'` or a CSS variable, not a Tailwind class.

### Risk 3: `#9333EA` ‚Üí `#0062F4` flattens loading/disabled state
- **PlayerFormSheet.tsx:551** had `saving ? '#9333EA' : '#6B21A8'` ‚Äî both now map to `#0062F4`, so the ternary can be simplified to just `'#0062F4'`.
